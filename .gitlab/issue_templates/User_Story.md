## User Story Template

### User Story
_As a_ [type of user]  
_I want_ [some goal or feature]  
_So that_ [some reason or benefit]

> Example:  
> As a **student**, I want to **receive notifications when my assignment is graded** so that **I can review feedback quickly.**

---

### Acceptance Criteria
_The story is complete when all the following are true:_

- [ ] Criterion 1: …
- [ ] Criterion 2: …
- [ ] Criterion 3: …

> Example:  
> - [ ] User receives an email notification within 5 minutes of grading.  
> - [ ] Notification includes assignment name and grade.  
> - [ ] Link directs user to the graded assignment page.

---

### Definition of Done (DoD)
_Before closing this issue, ensure the following are completed:_

- [ ] Code implemented and reviewed  
- [ ] Unit / integration tests added and passing  
- [ ] Documentation updated 
- [ ] Feature verified in local environment
